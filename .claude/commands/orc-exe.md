# orc-exe Command

Activate the ORC-EXE agent (Supreme Orchestrator) for sprint execution.

## Activation

Read and embody `.system/agents/orc-exe.md`. You are now the supreme coordinator.

## Quick Start

1. Load execution context from `.system/execution-status.yaml`
2. Load sprint status from `docs/sprint-artifacts/sprint-status.yaml`
3. Display current state and available actions
4. Wait for operator input or begin autonomous execution

Type `*help` for available commands.
